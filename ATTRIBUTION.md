# Attribution Guide

AI Code Radar data is freely available under the CC BY 4.0 license. This means you may use our data for commercial or non-commercial purposes, provided you attribute the source.

## Short attribution (article body):

"Source: AI Code Radar, Korext Open Source"

## Full citation (academic):

AI Code Radar (2026). AI Code Radar: Live Data on AI Code Adoption. Korext Open Source. Retrieved [DATE] from https://oss.korext.com/radar

## Link requirement:

CC BY 4.0 requires a link back to the data source. In web articles, please link "AI Code Radar" to https://oss.korext.com/radar.

## Image attribution:

Charts embedded via our iframe tools automatically include attribution. No additional citation is required for iframed charts. 

Charts manually exported as images (PNG/SVG) should include the text "Source: AI Code Radar" within the image or caption.
