# Originality Statement

AI Code Radar is an original work created by the Korext team.

The aggregation methodology, dashboard design, API schema, and data quality
controls are original implementations. No proprietary data, algorithms, or
infrastructure from any third party have been incorporated.

The statistical methods used (weighted percentages, Winsorization, k-anonymity
thresholds) are well established public domain techniques applied in a novel
context: measuring AI code adoption from publicly declared attestation data.

All source data comes from publicly available `.ai-attestation.yaml` files
that repository maintainers have voluntarily published.

No code in this repository is derived from or contains elements of any
proprietary Korext platform infrastructure.
